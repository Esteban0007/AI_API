How to Use?

  1. Sign Up (email + password)
  2. Confirm your email
  3. Copy your API Key from the dashboard
  4. Use the API (see documentation)

Example Usage

Basic API Request

curl -X POST "https://readyapi.net/api/v1/search/query" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "cute robot cleans the earth",
    "top_k": 5
  }'

📤 Step 1: Upload Documents

Send your documents as JSON: 

{
  "documents": [
    {
      "id": "part-001",
      "title": "Charge System",
      "content": "solar panels energy reactor battery charging power",
      "keywords": ["charge", "energy", "power"]
    },
    {
      "id": "part-002",
      "title": "Flight System",
      "content": "thruster engine fly speed navigate",
      "keywords": ["fly", "speed", "navigate"]
    }
  ]
}
curl -X POST \
  https://readyapi.net/api/v1/documents/upload \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d @documents.json

🔍 Step 2: Search with Natural Language

Example queries:

"I need to get somewhere far away quickly"
"Hide the ship so they can't find us"
"Fire all weapons at the enemy"

✅ Step 3: Get Ranked Results

Semantic search returns ranked matches: 

{
  "results": [
    {
      "id": "part-007",
      "title": "Warp System",
      "score": 0.48,
      "content": "warp jump teleport..."
    },
    {
      "id": "part-002",
      "title": "Flight System",
      "score": 0.46,
      "content": "thruster engine..."
    }
  ],
  "count": 2,
  "processing_ms": 124
}

💡 How it works: Upload JSON documents once. Then use natural language queries to find semantically similar content. The search engine understands meaning, not just keywords. Results include similarity scores from 0 to 1 (higher = better match).
