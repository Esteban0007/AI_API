📊 PIPELINE DE BÚSQUEDA
═════════════════════════════════════════════════════════════════════════════

    User Query
       ↓
    [Embedder] ─ Convertir query a vector
       ↓
    [Vector Search] ─ ANN busca top-10 similares
       ↓
    [Cross-Encoder] ─ Re-ranking para máxima relevancia
       ↓
    [Results] ─ Devuelve top-5 ordenados por score

🔌 API ENDPOINTS
═════════════════════════════════════════════════════════════════════════════

POST /api/v1/search/query
    └─ Búsqueda semántica
    └─ Input: {"query": "texto", "top_k": 5}
    └─ Output: [{"id": "...", "score": 0.95, ...}]

POST /api/v1/documents/upload
    └─ Cargar documentos
    └─ Input: {"documents": [...]}
    └─ Output: {"success": true, "uploaded_count": 10}

GET /api/v1/documents/stats
    └─ Estadísticas de la colección
    └─ Output: {"document_count": 150, "embedding_dimension": 384}

DELETE /api/v1/documents/{doc_id}
    └─ Eliminar documento

GET /api/v1/search/health
    └─ Verificar estado del servicio
